# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-30

### Added
- Initial release of terraform-aws-ec2 module
- EC2 instance provisioning with comprehensive configuration options
- Support for root and additional EBS volumes
- Elastic IP creation and attachment
- Custom network interface support
- IMDSv2 metadata security configuration
- User data script support
- Comprehensive tagging support
- Ubuntu Desktop example with RDP/VNC access
