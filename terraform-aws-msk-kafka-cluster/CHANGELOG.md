# Changelog

## [0.1.0] - 2026-04-08

### Added

- Initial release of the `terraform-aws-msk-kafka-cluster` module.
- Supports broker node group, encryption at rest / in transit, client authentication (SASL/IAM, SASL/SCRAM, TLS, unauthenticated), optional MSK configuration, CloudWatch broker logging, and Prometheus open monitoring.
